# Kartik Singhal - Portfolio Website

This is a personal portfolio website built with [Next.js](https://nextjs.org) and [Tailwind CSS](https://tailwindcss.com).

## Features

- **Hero Section**: Introduction and Call to Action.
- **About**: Profile summary and education.
- **Skills**: Technical skills categorization.
- **Projects**: Showcase of academic and personal projects.
- **Contact**: Functional contact form (logs to console/ready for API integration).
- **Responsive Design**: Optimized for all devices.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [kartik-singhal-portfolio](https://kartik-singhal-portfolio.onrender.com) with your browser to see the result.

## Deployment

This project is ready for deployment on [Render](https://render.com).
See `render.yaml` for configuration.

## Project Structure

- `app/`: Main application code (Pages, API).
- `components/`: Reusable UI components.
- `public/`: Static assets.
